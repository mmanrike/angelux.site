Markdown se puede decir que es una forma de escribir simple y sin distracciones, donde se permite dar formato al texto de una manera práctica y sencilla con unos marcadores especiales mientras se escribe, por ejemplo si queremos dar formato de negrita a un texto o frase, no necesitaremos seleccionar un texto y realizar una combinación de teclas (Ctrl + B o Ctrl + N) por ejemplo o mover e cursor hacia la barra de herramientas de nuestro procesador de texto (MS Word o Libreoffice o cualquier otro) y darle en el botón [N] para lograr el resultado deseado, todos estos pasos para realizar una negrita se puede decir que toma su tiempo y tiene un factor distractor, especialmente cuando se está redactando.

Con Markdown bastaría que cuando se esté escribiendo encerrar la palabra o texto requeridos en unos simples ** (asteriscos) para realizarlo, por ejemplo escribir así: **negrita** daría el siguiente resultado: negrita. Así como los asteriscos hay muchos otros códigos que se pueden utilizar como espacios, subguiones, etc. Para aprender la lista completa les recomiendo visitar MarkdownGuide (en inglés) o Markdown en español. Ahí se puede encontrar las guías definitivas. Por otra parte vale la pena mencionar que Markdown está presente en nuestras vidas cotidianas aún como usuario final, por ejemplo en el Whatsapp funcionan los códigos para Negrita, cursiva. Una implementación sencilla pero que puede ser de utilidad. Así como en el Whatsapp, para usuarios un poco mas avanzados Wordpress acepta esté código por defecto. Se puede integrar o escribir en el Gmail también y casi cualquier servicio social que haya gracias a las herramientas de exportación que los editores crean automáticamente para ti.

Ahora bien Markdown es una forma de escribir, no tiene todas las funcionalidades de un un procesador de textos y tampoco va a generar documentos tan completos, pero tiene todo lo necesario para usarlo en la vida digital, redactar correos, posts, etc. Se puede decir que es un intermedio entre ese editor de textos básicos que tienes (Notepad) y uno complejo como el Word. Y justamente hablando sobre editores Markdown, se han creado editores especiales de los cuales todos destacan por su simplicidad y poco uso de recursos.


Haroopad en Arcolinux. Imagen propia.

Los editores que he probado son:
  • Haroopad
  • Typora
Ambos son multi idioma y están disponibles de forma libre para Windows, Linux y OSx. Markdown también se integran con editores de textos populares mas avanzados como Sublime Text, Brackets, Atom, pero recomiendo usar los editores nativos por las características que ofrecen y sus respectivos beneficios.

En conclusión Markdown es una sencilla forma de escribir con una curva de aprendizaje fácil y rápida, es mas avanzado que un editor de texto, no tan complicado como el HTML ni tan pesado como para abrir un procesador de texto. Si escribir quieres para exportar a tus redes sociales, blogs, correos, etc. Markdown es para ti.
